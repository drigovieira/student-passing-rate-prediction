# Previsão de Aprovação de Alunos com Regressão Linear

Neste projeto, desenvolvi um modelo de regressão linear para prever o percentual de alunos aprovados em 2017, com base nos dados de aprovação dos anos de 2015 e 2016. A abordagem seguiu boas práticas de Ciência de Dados, englobando toda a jornada de análise exploratória, preparação dos dados, treinamento do modelo, e avaliação de desempenho, com o objetivo de gerar uma previsão robusta e precisa.

## Etapas do Projeto

### 1. Análise Exploratória dos Dados (EDA)
Iniciei com a compreensão do contexto e análise dos dados disponíveis, como os percentuais de aprovação de alunos em 2015 e 2016. Durante a EDA, explorei a correlação entre os anos e outras características importantes dos dados.

### 2. Preparação e Limpeza dos Dados
Realizei o pré-processamento necessário, incluindo a verificação de dados faltantes, transformação e normalização das variáveis. Também tratei a distribuição de dados para garantir um modelo robusto.

### 3. Modelagem de Regressão Linear
Utilizei a técnica de regressão linear simples para construir o modelo. A variável dependente foi o percentual de aprovação de 2016, com base nos dados de 2015. O modelo foi treinado, ajustado e avaliado usando métricas de performance apropriadas.

### 4. Avaliação do Modelo
Avaliei o desempenho do modelo através de gráficos de resíduos e análise dos resultados da regressão, buscando entender o ajuste do modelo e a acuracidade das previsões.

### 5. Previsões e Resultados
Finalmente, apliquei o modelo treinado para prever os percentuais de aprovação de 2017. Também analisei os erros de previsão, ajustando o modelo conforme necessário para melhorar a precisão.

## Resultados Obtidos

- O modelo de regressão linear mostrou uma boa correlação entre os dados de 2015 e 2016, permitindo uma previsão confiável do percentual de alunos aprovados em 2017.
- As previsões ajudaram a entender as tendências históricas de aprovação e a identificar possíveis áreas para intervenção no processo de ensino.

## Tecnologias Utilizadas

- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Statsmodels para análise de regressão
- Jupyter Notebook para execução e documentação do código

## Conclusão

Este projeto demonstra como técnicas de Ciência de Dados podem ser aplicadas para entender e prever resultados educacionais, oferecendo insights valiosos para a tomada de decisões em políticas educacionais.
